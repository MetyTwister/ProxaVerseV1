pv-assistant

pv-assistant, is a CLI tool, running 100% offline and locally. It is meant to be fast, efficient, and easy to use. Lightweight AI Model allows it to run on lower-end machines, such as laptops and sbc servers Installing via setup allows the user to choose which LLM to use, and customize how will the AI react and answer to prompts.
